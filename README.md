# didcj
A tool for running distributed codejam code

## didcj local

Run dcj locally using *dcj.sh*

Example:
`didcj local --nodes 10`

## didcj remote

Run dcj on remote

First start the nodes:
`didcj remote start --nodes 100`

Run:
`didcj remote --nodes 100`

At the end stop the nodes:
`didcj remote stop`
